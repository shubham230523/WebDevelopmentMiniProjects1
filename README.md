# WebDevelopmentMiniProjects1 🌐

A collection of 5 interactive mini-projects built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. These projects focus on creative UI/UX animations, DOM manipulation, and modern styling techniques.

---

## 🚀 Projects Overview

Each project in this repository explores a different aspect of frontend development:

### 1. 3D Boxes Background 📦
An interactive background featuring a grid of boxes that transform and animate into a 3D effect. 
* **Key Skills:** CSS Transforms, JavaScript DOM manipulation, and dynamic styling.

### 2. Form Input Wave 🌊
A sleek login/registration form where label characters "wave" up and down when an input field is focused.
* **Key Skills:** CSS Transitions, JavaScript string manipulation, and high-level UI feedback.

### 3. Rotating Navigation Animation 🔄
A unique navigation concept where the entire page content rotates to reveal a hidden circular menu.
* **Key Skills:** CSS Fixed Positioning, Z-index management, and complex CSS rotations.

### 4. Scroll Animation Project 📜
A dynamic content loader that animates elements into view as the user scrolls down the page.
* **Key Skills:** Intersection Observer API / Scroll Event Listeners, and CSS entry animations.

### 5. Hover Board 🎨
A high-performance interactive board where tiles light up with random colors when hovered over and slowly fade away.
* **Key Skills:** Event Delegation, JavaScript `Math.random()`, and CSS transitions for trailing effects.

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser (Chrome, Firefox, Edge, or Safari) to run these projects.

### Installation & Execution

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/shubham230523/WebDevelopmentMiniProjects1.git
    ```
2.  **Open a project:**
    Navigate to any project folder and open the `index.html` file in your browser.
    *Example:* `cd hover_board` -\> Open `index.html`
